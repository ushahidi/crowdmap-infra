Crowdmap
=========

Installs Crowdmap codebase

Requirements
------------

Assumes php, and nginx are already configured. Only installs and configures crowdmap codebase.

Role Variables
--------------

- crowdmap_base_path
- crowdmap_repo_url
- crowdmap_repo_version
- crowdmap_db_host
- crowdmap_db_user
- crowdmap_db_password
- crowdmap_db_name

Dependencies
------------

None.
